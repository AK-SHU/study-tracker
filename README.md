# study-tracker
A Study Tracker web app built using HTML, CSS, and JavaScript that helps users manage tasks, track study hours, maintain streaks, and visualize progress with charts.
Study Tracker is a web-based productivity application designed to help users manage their daily study routine efficiently. It allows users to create and track study tasks, monitor completed hours, and maintain consistency through a streak system.

The application includes features like task management (add, edit, delete), daily goal setting, progress visualization using charts, and a built-in Pomodoro timer to improve focus. It also offers a personalized experience with a simple login system and a visually appealing interface with light/dark mode.

By combining productivity tools with an intuitive UI, the project helps users build better study habits and stay motivated.
